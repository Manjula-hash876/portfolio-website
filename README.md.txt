# Personal Portfolio Website

## Project Overview

This project is a simple **Personal Portfolio Website** created using **HTML5** as part of a web development learning exercise and internship assignment. The website introduces basic web development concepts such as HTML structure, semantic elements, images, navigation links, and forms.

The portfolio displays personal information, skills, and a contact form that allows visitors to send messages.

---

## Objectives

The main objectives of this project are:

* Understand the basic structure of an HTML document
* Learn how to use HTML tags and attributes
* Create structured web pages using semantic HTML elements
* Implement internal navigation links
* Add images with alternative text
* Build a contact form with input fields and validation

---

## Technologies Used

* HTML5
* Basic CSS (optional styling)
* Visual Studio Code
* Web Browser (Chrome / Edge / Firefox)

## Project Structure

```
portfolio-website
│
├── index.html
├── style.css
├── README.md
│
└── images
    └── profile.jpg
```

### File Description

**index.html**
Contains the main structure of the portfolio website including header, navigation menu, sections, and contact form.

**style.css**
Provides styling for the webpage such as layout, colors, fonts, and spacing.

**images/**
Stores images used in the website such as the profile picture.

**README.md**
Provides documentation and explanation of the project.

---

## Website Sections

### Header

Displays the developer's name and navigation menu.

### About Section

Introduces the developer and includes a profile image and description.

### Skills Section

Shows a list of technical skills using HTML list elements.

### Contact Section

Contains a simple contact form with:

* Name
* Email
* Message

### Footer

Displays copyright information.

## How to Run the Project

Follow these steps to run the project:

1. Download or clone the repository.
2. Open the project folder in **Visual Studio Code**.
3. Locate the file **index.html**.
4. Double-click the file or open it in a web browser.

The portfolio website will load and display the webpage.
